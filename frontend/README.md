# Frontend - Yemma Solutions

Application React pour la plateforme de recrutement.

## Structure

```
frontend/
├── src/
│   ├── components/
│   ├── pages/
│   ├── services/
│   └── App.jsx
├── package.json
├── Dockerfile
└── README.md
```

## Développement

```bash
# Installer les dépendances
npm install

# Démarrer en mode développement
npm run dev
```

## Docker

```bash
# Build et démarrage
docker-compose up frontend
```
